About the project - Triangles Wall:

Previous Next
The goal of this project was to design a wall using triangle porcelain tiles. There was six different colors that would be randomly placed to build a 2 meters to 80 cm wall.
With python in a jupyter notebook, using matplotlib and numpy libraries, it was possible to create several samples in which there were selected two of them to be used as templates to the real one.
After merging and refining it with the triangle porcelain tiles on the ground, it was time to get it done in the living room.
The result you can see here, and the code for randomly create patterns you can find in my github repository.
