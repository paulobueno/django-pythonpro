Triangles Wall:

The goal of this project was to design a wall using triangle porcelain tiles. There were six different colors, randomly placed to build a 2.8x0.8 meters wall.

About the project:
With python in a jupyter notebook, using matplotlib and numpy libraries, it was possible to create several samples in which there were selected two of them to be used as templates to the real one. The source code can be found here bellow. I place the code in Google's Colaboratory so we can see the code running live. You can run by yourself and it will return a brand new wall randomly generated!
After merging and refining it with the triangle porcelain tiles on the ground, it was time to get it done in the living room.
Enjoy! :)
